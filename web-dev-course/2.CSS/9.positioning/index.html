Positioning

Div's are block elements


Default position = 'static'


Facebook is great.

FIXED position means the other elements will ignore the fixed element in the flow of the web page

Apple.

Relative positioning moves the element a specified amount relative to it's original position

following elements will still be positioned relative to the elements original position. NOT it's new position

Microsoft.

Absolute positioning positions the element relative to the top left of the page.

Positioned elements will show above other elements if those elements don't have a specified higher z-index.

With no index and no position. the last element will be on top.
