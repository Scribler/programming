Javascript Tips

  1. START with a status variable that your functions can relate to.
  2. Make a function for each action if possible
  3. For BUTTONS use:

    document.getElementById("sevens).onclick = function(){
    yourCodeHere;
    }

  4. If you want to use a counter you must make the interval accesible from outside the function. To do this follow these steps.
    1. First set an open variable. You will assign the interval too this later.

      var foo;

    2. In your function set the interval to 'foo'

      function bar(){
      foo = setInterval(function(){}, 1000)
      }

    3. Now you can use 'clearInterval' on the assigned variable "foo" to stop the interval even though you aren't in the scope of the counter's function.

      clearInterval(foo)


      For button:
      document.getElementById("sevens").onclick = function(){
      clearInterval(foo)
      }
